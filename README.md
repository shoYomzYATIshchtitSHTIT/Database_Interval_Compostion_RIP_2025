# Разработка интернет-приложений (РИП) 2025

## База данных частотности интервалов композиторов

Разработанное приложение позволит музыкальным аналитикам создавать и формировать композиции из выбранных ими интервалов на принадлежность произведения к классицизму, а музыкальным экспертам завершать или отклонять данные композиции.
